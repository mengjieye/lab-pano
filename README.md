# 实验室全景图展示网站

这是一个展示多个实验室全景图的网站，使用 Pannellum 全景图查看器实现。

## 功能特点

- 响应式设计，支持各种设备访问
- 分类展示不同实验室的全景图
- 支持全景图预览和交互式查看
- 支持自动旋转和指南针功能

## 部署说明

### 本地运行

1. 克隆仓库到本地
2. 使用 Python 启动本地服务器：
   ```bash
   python -m http.server 8000
   ```
3. 访问 http://localhost:8000

### 部署到 GitHub Pages

1. Fork 本仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 main 分支作为源
4. 等待几分钟后，网站将在 https://[你的用户名].github.io/[仓库名] 上线

## 文件结构

```
.
├── navigation.html    # 主导航页面
├── 物理实验室/       # 物理实验室全景图
├── 生物实验室/       # 生物实验室全景图
├── 化学实验室/       # 化学实验室全景图
└── 地理实验室/       # 地理实验室全景图
```

## 注意事项

- 确保所有全景图文件都已正确生成
- 检查所有链接路径是否正确
- 确保图片文件大小适中，避免加载过慢
